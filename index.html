HTML: Meta Tags

Utilizando a tag < meta> podemos fornecer informações relevantes sobre o conteúdo da nossa página a mecanismos de leitura de metadados de outras aplicações. Os metadados (informações sobre a própria aplicação) não são mostrados na página para o usuário.

Os elementos marcados com a tag < meta> são geralmente utilizados para especificar a descrição do site, informar o autor do documento, informar quando foi a última atualização do conteúdo, definir palavras-chave para sites de busca e indexação, entre outros.

Os metadados podem ser usados pelos navegadores (para saberem quando mostrar o conteúdo da página ou quando atualizar a página automaticamente, por exemplo), mecanismos de busca (através das palavras-chave) ou outros serviços web.

*As últimas versões do HTML5 permitiu o uso de uma meta tag para permitir mais controle para os web designers sobre as áreas visíveis para o usuário na página web.

Tipos de meta tags

  • Http-equiv

    O atributo http-equiv é usado basicamente para simular uma resposta HTTP.
    Ele não sobrescreve a configuração prévia do servidor, portanto só terá efeito se este não tiver o valor atribuído configurado no servidor de origem. É popularmente usado para dar refresh automaticamente em uma página e, antes da versão 5 do HTML, o valor content-type era utilizado para especificar o conjunto de caracteres a ser utilizado em uma página Web (Atualmente usa-se a metatag "meta charset = UTF-8"). Outros valores comumente usados com o htto-equiv incluem:

    • Content-Security-Policy 
                                              <meta http-equiv="Content-Security-Policy" content="default-src 'self' *.example.com">
    • Refresh 
                                              <meta http-equiv="refresh" content="30">
    • Default-Style 
                                              <meta http-equiv="default-style" content="/style.css">
